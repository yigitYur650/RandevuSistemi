Randevu Sistemi (Legacy Project) 
Bu proje, yazılım yolculuğumun ilk aşamalarında, yaklaşık 10 ay önce geliştirdiğim amatör bir randevu yönetim sistemidir. Gelişim sürecimi takip etmek ve nereden başladığımı unutmamak adına bu depoyu (repository) silmiyorum.

Proje Hakkında
Bu çalışma, temel web teknolojilerini ve basit bir backend yapısını öğrenirken ortaya çıkardığım ilk projelerden biridir. Kullanıcıların randevu alabileceği basit bir arayüz ve bu randevuları yöneten bir sunucu yapısından oluşur.

Teknolojiler
Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Veri Yönetimi: JSON / Yerel Dosya Sistemi

Neleri Öğrendim?
Bu projeyi yaparken şunları deneyimledim:

Temel HTTP istekleri (GET, POST) ve API mantığı.

DOM manipülasyonu ile dinamik içerik oluşturma.

Node.js ortamında sunucu ayağa kaldırma ve package.json yönetimi.

Basit CSS ile kullanıcı arayüzü tasarımı.

Yapay zeka kullanımında prompt verme konusunda deneyim edindim 

Backend yapısını bu projede ilk kez kullandım
